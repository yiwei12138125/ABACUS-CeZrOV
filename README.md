# ABACUS-CeZrOV
ABACUS Dataset: Oxygen Vacancy (OV) Formation Energies on CeZr Composite Oxide Surfaces This repository contains raw data regarding the Oxygen Vacancy (OV) defect formation energies on CeZr composite oxide surfaces, calculated using the ABACUS software package.
⚠️ Important Notes:
Data Status: The data provided here is in its raw state and has not been deeply organized or cleaned. It is intended for reference purposes only.
Disclaimer: Accuracy is not fully guaranteed; please use with caution.
